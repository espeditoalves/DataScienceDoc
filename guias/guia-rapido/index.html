Guia Rápido

Este é um guia rápido para começar a usar o MkDocs.

Passos Básicos

  1. Instale o MkDocs: pip install mkdocs.
  2. Crie um novo projeto: mkdocs new ..
  3. Inicie o servidor: mkdocs serve.
  4. Acesse http://localhost:8000 no navegador.
